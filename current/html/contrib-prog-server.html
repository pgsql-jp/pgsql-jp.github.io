G.2. サーバアプリケーション

<title>Server Applications</title>

Some applications run on the PostgreSQL server itself. Currently, no such applications are included in the contrib directory. See also PostgreSQLサーバアプリケーション for information about server applications that are part of the core PostgreSQL distribution.
