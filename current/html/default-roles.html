O.2. Default Roles Renamed to Predefined Roles

PostgreSQL 13 and below used the term 「Default Roles」. However, as these roles are not able to actually be changed and are installed as part of the system at initialization time, the more appropriate term to use is 「Predefined Roles」. See 22.5 for current documentation regarding Predefined Roles, and the release notes for PostgreSQL 14 for details on this change.
