36.12. column_column_usage

The view <literal>column_column_usage</literal> identifies all generated columns that depend on another base column in the same table. Only tables owned by a currently enabled role are included. column_column_usageビューは、同じテーブル内の別の基底列に基づくすべての生成列を識別します。 現在有効なロールが所有するテーブルのみが含まれます。
