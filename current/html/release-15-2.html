E.3. リリース15.2

<title>Release 15.2</title>

<title>Release date:</title> リリース日: 2023-02-09

This release contains a variety of fixes from 15.1. For information about new features in major release 15, see <xref linkend="release-15"/>. このリリースは15.1に対し、様々な不具合を修正したものです。 15メジャーリリースにおける新機能については、E.5を参照してください。

E.3.1. バージョン15.2への移行

<title>Migration to Version 15.2</title>

A dump/restore is not required for those running 15.X. 15.Xからの移行ではダンプ/リストアは不要です。

However, if you are upgrading from a version earlier than 15.1, see <xref linkend="release-15-1"/>. しかしながら、15.1より前のバージョンからアップグレードする場合は、E.4を参照してください。
