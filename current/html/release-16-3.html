E.1. リリース16.3 #

<title>Release 16.3</title>

<title>Release date:</title> リリース日: 2024-05-09

This release contains a variety of fixes from 16.2. For information about new features in major release 16, see <xref linkend="release-16"/>. 《機械翻訳》このリリースはさまざまな修正を含んでいます。 メジャーリリース16の新機能についてはE.4を参照してください。

E.1.1. バージョン16.3への移行 #

<title>Migration to Version 16.3</title>

A dump/restore is not required for those running 16.X. 《機械翻訳》16.X を実行している場合は、ダンプ/リストアは必要ありません。

However, a security vulnerability was found in the system views <structname>pg_stats_ext</structname> and <structname>pg_stats_ext_exprs</structname>, potentially allowing authenticated database users to see data they shouldn't. If this is of concern in your installation, follow the steps in the first changelog entry below to rectify it. 《機械翻訳》しかし、システムビューpg_stats_extとpg_stats_ext_exprsにセキュリティ上の脆弱性が見つかり、認証されたデータベースユーザが、本来見るべきでないデータを見ることができる可能性があります。 これがインストール環境で問題となる場合は、次の最初の変更ログの項目に従って修正してください。

Also, if you are upgrading from a version earlier than 16.2, see <xref linkend="release-16-2"/>. 《機械翻訳》また、16.2より前のバージョンからアップグレードする場合はE.2を参照してください。
