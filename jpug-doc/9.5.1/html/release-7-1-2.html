E.268.1. バージョン 7.1.2 への移行方法

<title>Migration to Version 7.1.2</title>

A dump/restore is <emphasis>not</emphasis> required for those running 7.1.X. 7.1.X からの移行の場合は ダンプ/リストアは必要ありません。

E.268.2. 変更点

<title>Changes</title> 
Fix PL/pgSQL SELECTs when returning no rows
Fix for psql backslash core dump
Referential integrity privilege fix
Optimizer fixes
pg_dump cleanups
