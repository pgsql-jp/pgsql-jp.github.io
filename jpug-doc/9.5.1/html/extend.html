In the sections that follow, we will discuss how you can extend the <productname>PostgreSQL</productname> <acronym>SQL</acronym> query language by adding: 本節では以下を追加することでPostgreSQLのSQL問い合わせ言語をどのように拡張できるかを説明します。

  • functions (starting in <xref linkend="xfunc">) 関数（「ユーザ定義関数」から）。

  • aggregates (starting in <xref linkend="xaggr">) 集約（「ユーザ定義の集約」から）。

  • data types (starting in <xref linkend="xtypes">) データ型（「ユーザ定義の型」から）。

  • operators (starting in <xref linkend="xoper">) 演算子（「ユーザ定義の演算子」から）。

  • operator classes for indexes (starting in <xref linkend="xindex">) インデックス用演算子クラス（「インデックス拡張機能へのインタフェース」から）。

  • packages of related objects (starting in <xref linkend="extend-extensions">) 関連オブジェクトのパッケージ（「関連するオブジェクトを拡張としてパッケージ化」から）。
