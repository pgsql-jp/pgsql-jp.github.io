名前

SPI_pop <refpurpose>pop SPI stack to return from recursive SPI usage</refpurpose> — 再帰的なSPIの使用から復帰できるようにSPIスタックをポップする

概要

void SPI_pop(void)

説明

<title>Description</title>

<function>SPI_pop</function> pops the previous environment from the SPI call stack. See <function>SPI_push</function>. SPI_popは、SPI呼び出しスタックから以前の環境をポップします。 SPI_pushを参照してください。
