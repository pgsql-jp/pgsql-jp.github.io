49.76. pg_tables

The view <structname>pg_tables</structname> provides access to useful information about each table in the database. pg_tablesビューはデータベース内のそれぞれのテーブルに関する有用な情報へのアクセスを提供します。

表49.77 pg_tablesの列

<title><structname>pg_tables</> Columns</title>
名前型参照先説明
schemanamenamepg_namespace.nspnameテーブルを持つスキーマ名
tablenamenamepg_class.relnameテーブル名

tablespacenamepg_tablespace.spcnameテーブルを含むテーブル空間の名前（データベースのデフォルトの場合はNULL）
