E.1. Release 9.5.1

Release Date

2016-02-11

This release contains a variety of fixes from 9.5.0. For information about new features in the 9.5 major release, see 「リリース9.5」.

E.1.1. Migration to Version 9.5.1

A dump/restore is not required for those running 9.5.X.

    In principle, both versions of PL/Python can be used in the same database, though not in the same session (because the two versions of libpython cannot safely be used concurrently).
