E.2. リリース12.3

<title>Release 12.3</title>

<title>Release date:</title> リリース日: 2020-05-14

This release contains a variety of fixes from 12.2. For information about new features in major release 12, see <xref linkend="release-12"/>. このリリースは12.2に対し、様々な不具合を修正したものです。 12メジャーリリースにおける新機能については、E.5を参照してください。

E.2.1. バージョン12.3への移行

<title>Migration to Version 12.3</title>

A dump/restore is not required for those running 12.X. 12.Xからの移行ではダンプ/リストアは不要です。

However, if you are upgrading from a version earlier than 12.2, see <xref linkend="release-12-2"/>. しかしながら、12.2より前のバージョンからアップグレードする場合には、E.3を参照してください。
