50.34. pg_operator

The catalog <structname>pg_operator</> stores information about operators. See <xref linkend="sql-createoperator"> and <xref linkend="xoper"> for more information. pg_operatorカタログは演算子の情報を保存します。 CREATE OPERATORと36.12. ユーザ定義の演算子を参照してください。

表50.34 pg_operatorの列
