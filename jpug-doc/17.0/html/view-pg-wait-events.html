52.36. pg_wait_events #

The view <structname>pg_wait_events</structname> provides description about the wait events. 《機械翻訳》ビューpg_wait_eventsは、待機イベントについての説明を提供します。

表52.36 pg_wait_events Columns

Column Type 列 型

Description 説明

type text

Wait event type 《機械翻訳》待機イベントのタイプ。

name text

Wait event name 《機械翻訳》待機イベント名。

description text

Wait event description 《マッチ度[54.545455]》説明 《機械翻訳》待機イベントの説明。
