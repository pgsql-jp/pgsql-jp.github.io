50.31. pg_largeobject_metadata

The catalog <structname>pg_largeobject_metadata</structname> holds metadata associated with large objects. The actual large object data is stored in <link linkend="catalog-pg-largeobject"><structname>pg_largeobject</></link>. pg_largeobject_metadataはラージオブジェクトに関連したメタデータを保持します。 実際のラージオブジェクトデータはpg_largeobjectに格納されます。
