VAR

VAR — 変数を定義します。

Synopsis

VAR varname IS ctype

説明

VARコマンドは新しいCデータ型にホスト変数を割り当てます。 ホスト変数は宣言セクションで前もって宣言されていなければなりません。

パラメータ

varname

Cの変数名です。

ctype

Cの型指定です。

例

Exec sql begin declare section;
short a;
exec sql end declare section;
EXEC SQL VAR a IS int;

互換性

VARコマンドはPostgreSQLの拡張です。
