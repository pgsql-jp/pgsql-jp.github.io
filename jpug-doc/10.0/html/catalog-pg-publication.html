51.40. pg_publication

The catalog <structname>pg_publication</structname> contains all publications created in the database. For more on publications see <xref linkend="logical-replication-publication">. カタログpg_publicationには、データベース内に作成されたすべてのパブリケーションが含まれます。 パルリケーションについての詳細は31.1を参照してください。

表51.40 pg_publicationの列

<title><structname>pg_publication</structname> Columns</title>
名前型参照先説明
oidoid 行識別子（隠し属性です。明示的に選択しなければなりません）
pubnamename パブリケーションの名前
pubowneroidpg_authid.oidパブリケーションの所有者
puballtablesbool 真の場合、このパブリケーションは、将来作成されるテーブルを含め、データベース内の全テーブルを自動的に含みます。
pubinsertbool 真の場合、パブリケーション内のテーブルに対するINSERTは複製されます。
pubupdatebool 真の場合、パブリケーション内のテーブルに対するUPDATEは複製されます。
pubdeletebool 真の場合、パブリケーション内のテーブルに対するDELETEは複製されます。
