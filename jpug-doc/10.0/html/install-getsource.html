16.3. ソースの入手

PostgreSQL 10.0のソースは、Webサイトhttps://www.postgresql.org/download/のダウンロードページから入手することができます。 postgresql-10.0.tar.gzまたはpostgresql-10.0.tar.bz2という名前のファイルを入手してください。 ファイルを入手したら、展開します。 

gunzip postgresql-10.0.tar.gz
tar xf postgresql-10.0.tar

（.bz2ファイルをダウンロードした場合はgunzipではなくbunzip2を使用してください。） これにより、カレントディレクトリ以下にpostgresql-10.0というディレクトリが作成され、PostgreSQLのソースが展開されます。 この後のインストール手順を行うために、このディレクトリに移動してください。

またバージョン管理システムのリポジトリから直接ソースを入手することができます。付録Iを参照してください。
