51.84. pg_sequences

The view <structname>pg_sequences</structname> provides access to useful information about each sequence in the database. ビューpg_sequencesはデータベース内の各シーケンスについての有用な情報へのアクセスを提供します。

表51.85 pg_sequencesの列
