51.43. pg_replication_origin

The <structname>pg_replication_origin</structname> catalog contains all replication origins created. For more on replication origins see <xref linkend="replication-origins">. pg_replication_originカタログは、作成されたすべてのレプリケーション起点を含んでいます。 レプリケーション起点についての詳細は第49章を参照してください。
