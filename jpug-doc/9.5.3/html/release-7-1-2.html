リリース日

E.278.1. バージョン7.1.2への移行

<title>Migration to Version 7.1.2</title>

A dump/restore is <emphasis>not</emphasis> required for those running 7.1.X. 7.1.Xからの移行の場合はダンプ/リストアは必要ありません。

E.278.2. 変更点

<title>Changes</title> 
Fix PL/pgSQL SELECTs when returning no rows
Fix for psql backslash core dump
Referential integrity privilege fix
Optimizer fixes
pg_dump cleanups
