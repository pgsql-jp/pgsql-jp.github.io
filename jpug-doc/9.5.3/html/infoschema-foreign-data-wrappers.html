34.26. foreign_data_wrappers

The view <literal>foreign_data_wrappers</literal> contains all foreign-data wrappers defined in the current database. Only those foreign-data wrappers are shown that the current user has access to (by way of being the owner or having some privilege). foreign_data_wrappersビューには現在のデータベース内で定義された、すべての外部データラッパーが含まれます。 現在のユーザが（所有者である、または、何らかの権限を持つことにより）アクセス可能な外部データラッパーのみが表示されます。
