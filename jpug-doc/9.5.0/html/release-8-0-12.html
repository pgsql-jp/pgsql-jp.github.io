E.187. リリース 8.0.12

<title>Release 8.0.12</title>

リリース日

<title>Release Date</title>

2007-02-07

This release contains one fix from 8.0.11. For information about new features in the 8.0 major release, see <xref linkend="release-8-0">.
このリリースには、8.0.11に対する1つの修正が含まれています。 8.0メジャーリリースにおける新機能については「リリース8.0」を参照してください。

E.187.1. バージョン8.0.12への移行

<title>Migration to Version 8.0.12</title>

A dump/restore is not required for those running 8.0.X. However, if you are upgrading from a version earlier than 8.0.6, see <xref linkend="release-8-0-6">.
8.0.Xからの移行ではダンプ/リストアは不要です。 しかし、8.0.6より前のバージョンからアップグレードする場合は、「リリース8.0.6」を参照してください。

E.187.2. 変更点

<title>Changes</title>

  • Remove overly-restrictive check for type length in constraints and functional indexes(Tom)
    制約および関数インデックスにおける型の長さに対する重複/限定検査を削除しました。(Tom)
