E.256. リリース7.3.16

<title>Release 7.3.16</title>

リリース日

<title>Release Date</title>

2006-10-16

This release contains a variety of fixes from 7.3.15. このリリースは7.3.15の各種不具合を修正したものです。

E.256.1. バージョン7.3.16への移行

<title>Migration to Version 7.3.16</title>

A dump/restore is not required for those running 7.3.X. However, if you are upgrading from a version earlier than 7.3.13, see <xref linkend="release-7-3-13">. 7.3.Xからの移行ではダンプ/リストアは不要です。 しかし、7.3.13より前のバージョンからアップグレードする場合は、E.259. リリース7.3.13を参照してください。

E.256.2. 変更点

<title>Changes</title>
<listitem><para>Fix corner cases in pattern matching for <application>psql</>'s <literal>\d</> commands</para></listitem>

  • psqlの\dコマンドに対するパターンマッチにおけるちょっとした部分を修正しました。

    • <listitem><para>Fix index-corrupting bugs in /contrib/ltree (Teodor)</para></listitem>

  • /contrib/ltreeにおけるインデックス破損不具合を修正しました。(Teodor)

    • <listitem><para>Back-port 7.4 spinlock code to improve performance and support 64-bit architectures better</para> </listitem>

  • 64bitアーキテクチャの性能とサポートをより向上させるために、7.4のスピンロックコードを移植しました。

    • <listitem><para>Fix SSL-related memory leak in libpq</para> </listitem>

  • libpqにおけるSSLに関したメモリリークを修正しました。

    • <listitem><para>Fix backslash escaping in /contrib/dbmirror</para></listitem>

  • /contrib/dbmirrorにおけるバックスラッシュエスケープ処理を修正しました。

    • <listitem><para>Adjust regression tests for recent changes in US DST laws </para> </listitem>

  • リグレッションテストをUS DST法の最近の変更に合わせました。
