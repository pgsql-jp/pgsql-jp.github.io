E.300. リリース7.0.1

<title>Release 7.0.1</title>

リリース日

<title>Release Date</title>

2000-06-01

This is a cleanup release for 7.0. このリリースは 7.0 を整理したものです。

E.300.1. バージョン7.0.1への移行

<title>Migration to Version 7.0.1</title>

A dump/restore is <emphasis>not</emphasis> required for those running 7.0. 7.0からの移行の場合はダンプ/リストアは必要ありません。
