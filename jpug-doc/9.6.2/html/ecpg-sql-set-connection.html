名前

SET CONNECTION <refpurpose>select a database connection</refpurpose> — データベース接続を選択します。

概要

SET CONNECTION [ TO | = ] connection_name

説明

<title>Description</title>

<command>SET CONNECTION</command> sets the <quote>current</quote> database connection, which is the one that all commands use unless overridden. SET CONNECTIONは、上書きされない限りすべてのコマンドが使用する、「現在の」データベース接続を設定します。

パラメータ

<title>Parameters</title>
connection_name

A database connection name established by the <command>CONNECT</command> command. CONNECTコマンドで確立したデータベース接続の名前です。

DEFAULT

Set the connection to the default connection. 接続をデフォルトの接続に設定します。

例

<title>Examples</title> 
EXEC SQL SET CONNECTION TO con2;
EXEC SQL SET CONNECTION = con1;

互換性

<title>Compatibility</title>

<command>SET CONNECTION</command> is specified in the SQL standard. SET CONNECTIONは標準SQLで規定されています。
